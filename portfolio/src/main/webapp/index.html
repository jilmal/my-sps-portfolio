Web Development

Housing Justice Evaluation Tool

Currently for my Junior Design Project, my team and I have been working alongside a client to create an application for housing experts to more effectively counsel African American communities suffering from a lack of housing justice. These communties are suffering from a vast number of issues including gentrification, home abandonment, mortgage crisis, etc. The platform allows counselors to quickly assess homeowner concerns and turn over solutions immediatly on a case by case basis.

UI Design

Fortnite Track 2.0

My team and I researched and developed the UI for a Fortnite Esports platform targeted to help streamline the process for novice players to become pro players. The main feature of the application highlights users' "problem areas" (i.e. poor aim performance) and provides training videos and guides to improve the player's skill.

Web Development

Coorganate

During my internship at IBM, my team and I created a web application for physicians and transplant centers to mitigate organ waste currently happening in hospital systems. I worked alongside 2 other peers to implement the frontend of the platform utilizing HTML, CSS, and Materialze.

Mbed/ARM Development

YeeYee Adventures

During my Programming for Hardware/Software Systems course, I created a cowboy adventure video game utilizing a Hash Table implementation in C to render the game objects, maps, and sprites. The game was ran on an Mbed microcontroller which allowed the users to navigate the game through motion detection and interact with objects through external buttons.
